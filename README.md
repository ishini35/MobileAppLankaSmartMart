# 🍽️ Smart Restaurant Menu with AI Agent

> **“Scan, chat, and order – your digital waiter on demand.”**

---

## 🚀 Project Overview

Smart Restaurant Menu is a mobile-friendly web application that enhances the dining experience using an AI-powered chatbot 🤖.

Customers can scan a QR code, explore the menu, ask questions, and place orders directly through an interactive chat interface.

---

## 🌐  Try the live app below

👉 https://smart-restaurant-menu-frontend.onrender.com  

---

## 💡 Problem Statement

Traditional restaurant menus are not interactive.  
Customers often:
- Wait for waiters  
- Struggle to choose food based on preferences  
- Cannot easily ask questions about dishes  

---

## ✅ Solution

We developed a **QR-based smart menu system** where users can:
- Scan and access the menu instantly  
- Interact with an AI chatbot  
- Get personalized food recommendations  
- Place orders(easily)

---

## 🛠️ Technologies Used

- **React** – Frontend development  
- **Google Dialogflow** – AI chatbot integration  
- **Firebase** – Database and backend services  

---

## ✨ Key Features

- 📱 QR code-based menu access  
- 🤖 AI chatbot for food recommendations  
- ❓ Answers questions like *“Is this vegetarian?”*  
- 🌶️ Smart suggestions (spicy, healthy, popular dishes)  
- 🛒 Add to cart & place orders via chatbot  
- 📲 Mobile-friendly responsive design
